Directory List:

1. Arduino-Code: the code of receiving and sending infrared signal.

2. IDE and SDK: the software of Arduino IDE and Microsoft Kinect SDK.

3. InfraredDeviceDatasheet: the Datasheet of Infrared Receiver and Infrared LED.

4. KinectTV-Code: the code of the application KinectTV.

5. Libraries: the libraries that must be used in this project.

6. TestCode: test codes that I developed to test the functionalities in Microsoft Kinect SDK.

7. SetUpInstruction: the process of set up.
